There are six heading levels from h1 (the highest) to h6 (the lowest)

When a heading is nested in a document banner '<'header'>', it is the heading for the application or site

When nested in '<'main'>', whether or not it is nested within a '<'header'>' in '<'main'>', it is the header for that page, not the whole site

When nested in an '<'article'>' or '<'section'>', it is the header for that subsection of the page
Headings usage notes are as follows:
Heading information can be used by user agents to construct a table of contents for a document automatically
Do not use heading elements to resize text. Instead, use the CSS font-size property
Do not skip heading levels: always start from '<'h1'>', followed by '<'h2'>' and so on
